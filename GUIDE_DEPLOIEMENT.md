# 🚀 Guide de Déploiement Public : Landing Page DOMOYA

Ce guide vous présente les méthodes gratuites et sécurisées les plus simples pour mettre en ligne la landing page **DOMOYA** de prestige que nous venons de coder, afin d'obtenir immédiatement l'**URL Web publique** demandée pour votre candidature.

---

## ⚡ Méthode A : Netlify Drop (La plus simple et rapide - Recommandée)

**Temps estimé :** Moins de 10 secondes  
**Aucune installation, aucun compte préalable requis pour le premier jet.**

1. Ouvrez votre navigateur internet et rendez-vous sur : [Netlify Drop](https://app.netlify.com/drop)
2. Ouvrez l'explorateur de fichiers de votre ordinateur au chemin suivant :
   `C:\Users\original\Desktop\UNIR\OIF_CAP_INNOVANT`
3. Sélectionnez les fichiers suivants :
   - `index.html` (La landing page)
   - `african_modern_villa.png` (La visualisation 3D de la villa)
4. **Glissez-déposez** ces deux fichiers directement dans la zone de dépôt grise au centre de la page Netlify Drop.
5. Netlify déploie instantanément votre site et vous fournit une **URL sécurisée HTTPS** (ex. `https://unir-domoya.netlify.app` ou un nom aléatoire que vous pouvez personnaliser).

*Note : Vous pouvez ensuite créer un compte Netlify gratuit en un clic pour conserver cette URL de manière permanente et modifier son nom à votre convenance.*

---

## 🌌 Méthode B : Vercel (Hébergement Premium à Haute Performance)

**Temps estimé :** 1 minute  
**Idéal pour une infrastructure mondiale à chargement instantané.**

1. Rendez-vous sur : [Vercel](https://vercel.com) et créez un compte gratuit (Hobby).
2. Cliquez sur **Add New > Project**.
3. Vous pouvez y glisser votre dossier local ou connecter votre dépôt GitHub s'il est déjà synchronisé.
4. L'outil détectera automatiquement le fichier `index.html` et configurera le déploiement mondial en quelques secondes.

---

## 🛠️ Méthode C : GitHub Pages (Intégration Git Directe)

**Temps estimé :** 2 minutes  
**Si votre dossier de projet est déjà lié à un dépôt GitHub public :**

1. Poussez les modifications sur votre dépôt avec les commandes suivantes dans votre terminal :
   ```bash
   git add index.html african_modern_villa.png
   git commit -m "feat: add premium smart housing landing page"
   git push origin main
   ```
2. Sur l'interface GitHub de votre dépôt, allez dans l'onglet **Settings** (Paramètres).
3. Dans la barre latérale gauche, cliquez sur **Pages** (sous la section *Code and automation*).
4. Sous **Build and deployment > Source**, choisissez **Deploy from a branch**.
5. Sous **Branch**, sélectionnez votre branche principale (`main` ou `master`), laissez le dossier à la racine (`/`) et cliquez sur **Save**.
6. GitHub Pages générera votre lien public sécurisé dans la minute qui suit (ex. `https://votre-nom.github.io/votre-depot`).
